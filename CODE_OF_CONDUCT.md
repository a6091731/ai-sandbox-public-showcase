# Code of Conduct

## Our Standard

This repository is a public technical showcase. We expect respectful,
constructive, and professional communication in all interactions.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language
- Sharing private customer, product, or infrastructure information
- Posting secrets, credentials, tokens, or internal-only materials

## Enforcement

Maintainers may remove or edit content that violates these standards and
may restrict participation for repeated or severe violations.

## Reporting

For sensitive concerns, use private security reporting via `SECURITY.md`.
For general conduct concerns, open an issue with minimal necessary context.
